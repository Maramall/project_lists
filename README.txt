Как всем этим пользоваться:
1. скачать все файлы, кроме этого, в одну папку
2. открыть консоль и перейти в директорию, в которой находятся скачанные файлы
3. прописать:
$ make
4. после выполнения появится файл main,который нужно запустить:
$ ./main
готово
краткое описание этой программы:
в файле input.txt находится некоторый текст, который обрабатывается программой, а именно происходит поиск в тексте 100 популярных слов.
