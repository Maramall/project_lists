Как всем этим пользоваться:
0. использовать UNIX-подобную операционную систему
1. скачать все файлы, кроме этого, в одну папку
2. открыть консоль и перейти в директорию, в которой находятся скачанные файлы
3. прописать:
$ make -f name
где name это Makemain или Makeklass
4. после выполнения появится файл main,который нужно запустить:
$ ./main #если выбрали Makemain 
или
$ ./klass #если выбрали Makeklass
готово
краткое описание программы main:
в файле input.txt находится некоторый текст, который обрабатывается программой, а именно происходит поиск в тексте 100 популярных слов, после чего список этих слов записывается в файл output.txt
краткое описание программы klass:
создается список класса, с которым можно манипулировать тремя командами: поиск ученика по фамилии, удаление ученика по фамилии и вывод всего списка
